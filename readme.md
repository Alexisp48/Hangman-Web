

## -- Hangman Web -- ##


# - Présentation - #

Hangman web est un project de hangman en ligne.

Le project utilise du HTML / CSS / GO

# - The 8th era - #

Le but du jeu qui s'appelle "The 8th era" est d'arriver au 8eme ère.

On peut passer à l'ère suivante contre une certaine somme d'or qui augemente au fur et à meusure des âges. 
Il faut aussi s'occuper des habiant de son village car il a un system de nouriture qui diminue toutes les 20s.
On peut aussi acheter de la nouriture contre de l'or.
Enfin le centre du jeu est la mine d'or qui est la page de hangman si on gagne la partie de hangman on gagne de l'or pour avancer dans le jeu.
